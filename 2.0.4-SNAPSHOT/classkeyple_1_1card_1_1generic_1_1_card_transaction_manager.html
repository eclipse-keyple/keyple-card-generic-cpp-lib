◆ processApdusToByteArrays()

virtual const std::vector< std::vector< uint8_t > > keyple::card::generic::CardTransactionManager::processApdusToByteArrays ( )
pure virtual

Transmits all prepared APDUs, closes the physical channel if required, and returns a list of responses to the APDUs in the form of list of of const uint8_t arrays.

If the prepared APDU list is empty an empty list is returned.

The prepared APDU list is cleared after this method has been invoked.

Returns
A not null reference.
Exceptions
TransactionExceptionIf the communication with the card or the reader has failed.
